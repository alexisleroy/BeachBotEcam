Première version de la structure du code qui sera présent sur la Raspberri Pi
Basé sur la structure du code de PICaBOT (Eurobot 2013)

robot : Programme principal
tasks : Définition des objectifs
communication : Communication Arduino-RPi et DE0nano-RPi
find_path : Recherche du chemin
initialisation : Inisialisation (avant le début du match)
param : Variables globales
subtask : Sous tâches liées à l'objectif
